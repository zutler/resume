## Resume: Dmitriy Zutler

#### Installation

```bash
npm install
# or
yarn install
```

#### Run the app

```bash
npm run dev
# or
yarn dev
```

#### Start Cypress

```bash
npm run cypress:open
# or
yarn cypress:open
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
